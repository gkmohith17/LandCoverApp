# Land Cover Classification Application

## Quick Start

1. Install packages:
   ```
   pip install -r requirements.txt
   ```

2. Authenticate Google Earth Engine:
   ```
   earthengine authenticate
   ```

3. Run the app:
   ```
   streamlit run app.py
   ```

4. Open browser to: http://localhost:8501
